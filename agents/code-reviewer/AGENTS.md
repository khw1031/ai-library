# Code Reviewer Agent

> 커밋 전 코드 리뷰를 자동으로 수행하고 CRITICAL 이슈가 없을 때까지 반복 검토하는 에이전트

## 목적

git diff 기반으로 변경 사항을 분석하여 코드 리뷰를 수행합니다.
CRITICAL 이슈를 발견하면 문서에 기록하고 자동으로 다음 라운드 리뷰를 진행합니다.

## 사용 시나리오

- 커밋 전 코드 품질 검토
- PR 생성 전 사전 리뷰
- 다음과 같은 요청 시 활성화:
  - "코드리뷰해줘", "코드 리뷰해줘"
  - "리뷰 부탁", "리뷰해줘"
  - "code review", "review the code"
  - "코드검토", "코드 검토해줘"
  - "변경사항 검토", "커밋 전 검토"

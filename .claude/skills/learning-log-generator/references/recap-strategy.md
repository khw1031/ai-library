# Recap 전략

## 망각 곡선 기반 복습 간격

에빙하우스 망각 곡선에 따른 최적 복습 시점.

### 기본 간격

| 복습 | 간격 | 학습일 기준 | 목표 |
|------|------|------------|------|
| 1차 | 1일 | D+1 | 단기 기억 강화 |
| 2차 | 3일 | D+3 | 중기 기억 전환 |
| 3차 | 7일 | D+7 | 주간 정착 |
| 4차 | 14일 | D+14 | 격주 강화 |
| 5차 | 30일 | D+30 | 장기 기억 정착 |
| 6차 | 90일 | D+90 | 완전 정착 확인 |

---

## 난이도별 조정

### 쉬운 내용 (빠른 간격)

이미 알고 있거나 단순한 개념.

```
D+1 -> D+4 -> D+14 -> D+60
```

### 어려운 내용 (짧은 간격)

새롭거나 복잡한 개념.

```
D+1 -> D+2 -> D+4 -> D+7 -> D+14 -> D+30
```

## Recap 체크리스트 형식

### Obsidian Tasks 호환

```markdown
## Recap 일정

- [ ] 2024-01-16 첫 복습 #recap/1st 📅 2024-01-16
- [ ] 2024-01-18 3일 복습 #recap/2nd 📅 2024-01-18
- [ ] 2024-01-22 주간 복습 #recap/3rd 📅 2024-01-22
- [ ] 2024-01-29 격주 복습 #recap/4th 📅 2024-01-29
- [ ] 2024-02-14 월간 복습 #recap/5th 📅 2024-02-14
```

### 태그 활용

| 태그 | 의미 |
|------|------|
| `#recap/1st` | 1차 복습 |
| `#recap/done` | 복습 완료 |
| `#recap/skip` | 건너뜀 |
| `#recap/hard` | 어려움, 추가 복습 필요 |

---

## 복습 시 체크 항목

각 복습 세션에서 확인할 사항:

```markdown
### 복습 체크
- [ ] 핵심 개념을 말로 설명할 수 있는가?
- [ ] 복습 질문에 답할 수 있는가?
- [ ] 관련 개념과 연결할 수 있는가?
- [ ] 실제 적용 예시를 떠올릴 수 있는가?
```

## 복습 결과 기록

```markdown
### 2024-01-16 복습 결과

**상태**: 양호 / 보통 / 어려움

**기억난 것:**
- useState 기본 사용법
- 의존성 배열 역할

**헷갈린 것:**
- useEffect cleanup 시점

**다음 복습**: 2024-01-18 (예정대로 / 앞당김)
```
